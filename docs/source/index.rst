Ward - A modern Python test framework
=====================================

Installation
^^^^^^^^^^^^
Run ``pip install ward``. Ward requires Python 3.6+ in order to run.

Usage
^^^^^
You can run all tests in your current directory by just running ``ward`` with no arguments.
